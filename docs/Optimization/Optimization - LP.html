Linear Programming

Formally, a linear program is an optimization problem of the form

minimize⁡  c⊤x subject to Ax=bx≥0,\begin{aligned} \operatorname{minimize} \; & \boldsymbol{c}^{\top} \boldsymbol{x} \\ \text { subject to } & \boldsymbol{A} \boldsymbol{x}=\boldsymbol{b} \\ & \boldsymbol{x} \geq \mathbf{0}, \end{aligned}minimize subject to ​c⊤xAx=bx≥0,​

where c∈Rn,b∈Rm\boldsymbol{c} \in \mathbb{R}^n, \boldsymbol{b} \in \mathbb{R}^mc∈Rn,b∈Rm, and A∈Rm×n\boldsymbol{A} \in \mathbb{R}^{m \times n}A∈Rm×n. The vector inequality x≥0\boldsymbol{x} \geq \mathbf{0}x≥0 means that each component of x\boldsymbol{x}x is nonnegative. Several variations of this problem are possible; for example, instead of minimizing, we can maximize, or the constraints may be in the form of inequalities, such as Ax≥b\boldsymbol{A} \boldsymbol{x} \geq \boldsymbol{b}Ax≥b or Ax≤b\boldsymbol{A} \boldsymbol{x} \leq \boldsymbol{b}Ax≤b. Here A\boldsymbol{A}A is an m×nm \times nm×n matrix composed of real entries, m<n,rank⁡A=mm<n, \operatorname{rank} \boldsymbol{A}=mm<n,rankA=m.

Standard Form Linear Programs​

If a linear program is in the form

minimize⁡  c⊤x subject to Ax≥bx≥0\begin{aligned} \operatorname{minimize} \; & \boldsymbol{c}^{\top} \boldsymbol{x} \\ \text { subject to } & \boldsymbol{A} \boldsymbol{x} \geq \boldsymbol{b} \\ & \boldsymbol{x} \geq \mathbf{0} \end{aligned}minimize subject to ​c⊤xAx≥bx≥0​

then by introducing surplus variables yiy_iyi​, we can convert the original problem into the standard form In more compact notation, the formulation above can be represented as minimize⁡c⊤x\operatorname{minimize} \boldsymbol{c}^{\top} \boldsymbol{x}minimizec⊤x subject to Ax−Imy=[A,−Im][xy]=b\boldsymbol{A} \boldsymbol{x}-\boldsymbol{I}_m \boldsymbol{y}=\left[\boldsymbol{A},-\boldsymbol{I}_m\right]\left[\begin{array}{l}\boldsymbol{x} \\ \boldsymbol{y}\end{array}\right]=\boldsymbol{b}Ax−Im​y=[A,−Im​][xy​]=b

x≥0,y≥0,\boldsymbol{x} \geq \mathbf{0}, \boldsymbol{y} \geq \mathbf{0},x≥0,y≥0,

where Im\boldsymbol{I}_mIm​ is the m×mm \times mm×m identity matrix. If, on the other hand, the constraints have the form

Ax≤bx≥0,\begin{aligned} \boldsymbol{A} \boldsymbol{x} & \leq \boldsymbol{b} \\ \boldsymbol{x} & \geq \mathbf{0}, \end{aligned}Axx​≤b≥0,​

then we introduce slack variables yiy_iyi​ to convert the constraints into the form

Ax+Imy=[A,Im][xy]=bx≥0,y≥0\begin{aligned} &\boldsymbol{A} \boldsymbol{x}+\boldsymbol{I}_m \boldsymbol{y}=\left[\boldsymbol{A}, \boldsymbol{I}_m\right]\left[\begin{array}{c} \boldsymbol{x} \\ \boldsymbol{y} \end{array}\right]=\boldsymbol{b} \\ &\boldsymbol{x} \geq \mathbf{0}, \boldsymbol{y} \geq \mathbf{0} \end{aligned}​Ax+Im​y=[A,Im​][xy​]=bx≥0,y≥0​

where y\boldsymbol{y}y is the vector of slack variables. Note that neither surplus nor slack variables contribute to the objective function c⊤x\boldsymbol{c}^{\top} \boldsymbol{x}c⊤x.

Basic Solutions​

Definition​

We call [xB⊤,0⊤]⊤\left[\boldsymbol{x}_B^{\top}, \boldsymbol{0}^{\top}\right]^{\top}[xB⊤​,0⊤]⊤ a basic solution to Ax=b\boldsymbol{A x}=\boldsymbol{b}Ax=b with respect to the basis B\boldsymbol{B}B. We refer to the components of the vector xB\boldsymbol{x}_BxB​ as basic variables and the columns of B\boldsymbol{B}B as basic columns.

If some of the basic variables of a basic solution are zero, then the basic solution is said to be a degenerate basic solution. A vector x\boldsymbol{x}x satisfying Ax=b,x≥0\boldsymbol{A} \boldsymbol{x}=\boldsymbol{b}, \boldsymbol{x} \geq \mathbf{0}Ax=b,x≥0, is said to be a feasible solution. A feasible solution that is also basic is called a basic feasible solution. If the basic feasible solution is a degenerate basic solution, then it is called a degenerate basic feasible solution.

Properties of Basic Solutions​

Definition​

Any vector x\boldsymbol{x}x that yields the minimum value of the objective function c⊤x\boldsymbol{c}^{\top} \boldsymbol{x}c⊤x over the set of vectors satisfying the constraints Ax=b,x≥0\boldsymbol{A} \boldsymbol{x}=\boldsymbol{b}, \boldsymbol{x} \geq \mathbf{0}Ax=b,x≥0, is said to be an optimal feasible solution.

An optimal feasible solution that is basic is said to be an optimal basic feasible solution.

Fundamental Theorem of LP​

Consider a linear program in standard form.

  1. If there exists a feasible solution, then there exists a basic feasible solution.
  2. If there exists an optimal feasible solution, then there exists an optimal basic feasible solution.

Proof​

We first prove part 1. Suppose that x=[x1,…,xn]⊤\boldsymbol{x}=\left[x_1, \ldots, x_n\right]^{\top}x=[x1​,…,xn​]⊤ is a feasible solution and it has ppp positive components. Without loss of generality, we can assume that the first ppp components are positive, whereas the remaining components are zero. Then, in terms of the columns of A=[a1,…,ap,…,an]\boldsymbol{A}=\left[\boldsymbol{a}_1, \ldots, \boldsymbol{a}_p, \ldots, \boldsymbol{a}_n\right]A=[a1​,…,ap​,…,an​], this solution satisfies

x1a1+x2a2+⋯+xpap=b.x_1 \boldsymbol{a}_1+x_2 \boldsymbol{a}_2+\cdots+x_p \boldsymbol{a}_p=\boldsymbol{b} .x1​a1​+x2​a2​+⋯+xp​ap​=b.

There are now two cases to consider.

Case 1: If a1,a2,…,ap\boldsymbol{a}_1, \boldsymbol{a}_2, \ldots, \boldsymbol{a}_pa1​,a2​,…,ap​ are linearly independent, then p≤mp \leq mp≤m. If p=mp=mp=m, then the solution x\boldsymbol{x}x is basic and the proof is done. If, on the other hand, p<mp<mp<m, then, since rank⁡A=m\operatorname{rank} \boldsymbol{A}=mrankA=m, we can find m−pm-pm−p columns of A\boldsymbol{A}A from the remaining n−pn-pn−p columns so that the resulting set of mmm columns forms a basis. Hence, the solution x\boldsymbol{x}x is a (degenerate) basic feasible solution corresponding to the basis above.

Case 2: Assume that a1,a2,…,ap\boldsymbol{a}_1, \boldsymbol{a}_2, \ldots, \boldsymbol{a}_pa1​,a2​,…,ap​ are linearly dependent. Then, there exist numbers yi,i=1,…,py_i, i=1, \ldots, pyi​,i=1,…,p, not all zero, such that

y1a1+y2a2+⋯+ypap=0.y_1 \boldsymbol{a}_1+y_2 \boldsymbol{a}_2+\cdots+y_p \boldsymbol{a}_p=\mathbf{0} .y1​a1​+y2​a2​+⋯+yp​ap​=0.

We can assume that there exists at least one yiy_iyi​ that is positive, for if all the yiy_iyi​ are nonpositive, we can multiply the equation above by −1-1−1. Multiply the equation by a scalar ε\varepsilonε and subtract the resulting equation from x1a1+x2a2+x_1 a_1+x_2 \boldsymbol{a}_2+x1​a1​+x2​a2​+ ⋯+xpap=b\cdots+x_p \boldsymbol{a}_p=\boldsymbol{b}⋯+xp​ap​=b to obtain

(x1−εy1)a1+(x2−εy2)a2+⋯+(xp−εyp)ap=b.\left(x_1-\varepsilon y_1\right) \boldsymbol{a}_1+\left(x_2-\varepsilon y_2\right) \boldsymbol{a}_2+\cdots+\left(x_p-\varepsilon y_p\right) \boldsymbol{a}_p=\boldsymbol{b} .(x1​−εy1​)a1​+(x2​−εy2​)a2​+⋯+(xp​−εyp​)ap​=b.

Let

Then, for any ε\varepsilonε we can write

A[x−εy]=b.\boldsymbol{A}[\boldsymbol{x}-\varepsilon \boldsymbol{y}]=\boldsymbol{b} .A[x−εy]=b.

Let ε=min⁡{xi/yi:i=1,…,p,yi>0}\varepsilon=\min \left\{x_i / y_i: i=1, \ldots, p, y_i>0\right\}ε=min{xi​/yi​:i=1,…,p,yi​>0}. Then, the first ppp components of x−εy\boldsymbol{x}-\varepsilon \boldsymbol{y}x−εy are nonnegative, and at least one of these components is zero. We then have a feasible solution with at most p−1p-1p−1 positive components. We now prove part 2. Suppose that x=[x1,…,xn]⊤\boldsymbol{x}=\left[x_1, \ldots, x_n\right]^{\top}x=[x1​,…,xn​]⊤ is an optimal feasible solution and only the first ppp variables are nonzero.
