Gradient Descent

Suppose that we are given a point x(k)\boldsymbol{x}^{(k)}x(k). To find the next point x(k+1)\boldsymbol{x}^{(k+1)}x(k+1), we start at x(k)\boldsymbol{x}^{(k)}x(k) and move by an amount −αk∇f(x(k))-\alpha_k \nabla f\left(\boldsymbol{x}^{(k)}\right)−αk​∇f(x(k)), where αk\alpha_kαk​ is a positive scalar called the step size. We refer to this as a gradient descent algorithm (or simply a gradient algorithm).
