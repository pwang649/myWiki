Conjugate Direction Methods

The class of conjugate direction methods can be viewed as being intermediate between the method of steepest descent and Newton's method. The conjugate direction methods have the following properties:

  1. Solve quadratics of nnn variables in nnn steps.
  2. The usual implementation, the conjugate gradient algorithm, requires no Hessian matrix evaluations.
  3. No matrix inversion and no storage of an n×nn \times nn×n matrix are required. The conjugate direction methods typically perform better than the method of steepest descent, but not as well as Newton's method. As we saw from the method of steepest descent and Newton's method, the crucial factor in the efficiency of an iterative search method is the direction of search at each iteration. For a quadratic function of nnn variables f(x)=12x⊤Qx−x⊤bf(\boldsymbol{x})=\frac{1}{2} \boldsymbol{x}^{\top} \boldsymbol{Q} \boldsymbol{x}-\boldsymbol{x}^{\top} \boldsymbol{b}f(x)=21​x⊤Qx−x⊤b, x∈Rn,Q=Q⊤>0\boldsymbol{x} \in \mathbb{R}^n, \boldsymbol{Q}=\boldsymbol{Q}^{\top}>0x∈Rn,Q=Q⊤>0, the best direction of search, as we shall see, is in the Q\boldsymbol{Q}Q-conjugate direction. Basically, two directions d(1)\boldsymbol{d}^{(1)}d(1) and d(2)\boldsymbol{d}^{(2)}d(2) in Rn\mathbb{R}^nRn are said to be Q\boldsymbol{Q}Q-conjugate if d(1)⊤Qd(2)=0\boldsymbol{d}^{(1) \top} \boldsymbol{Q} \boldsymbol{d}^{(2)}=0d(1)⊤Qd(2)=0. The Conjugate Gradient Algorithm​

The conjugate gradient algorithm does not use prespecified conjugate directions, but instead computes the directions as the algorithm progresses. At each stage of the algorithm, the direction is calculated as a linear combination of the previous direction and the current gradient, in such a way that all the directions are mutually QQQ-conjugate - hence the name conjugate gradient algorithm. This calculation exploits the fact that for a quadratic function of nnn variables, we can locate the function minimizer by performing nnn searches along mutually conjugate directions.
As before, we consider the quadratic function

f(x)=12x⊤Qx−x⊤b,x∈Rnf(\boldsymbol{x})=\frac{1}{2} \boldsymbol{x}^{\top} \boldsymbol{Q} \boldsymbol{x}-\boldsymbol{x}^{\top} \boldsymbol{b}, \quad \boldsymbol{x} \in \mathbb{R}^nf(x)=21​x⊤Qx−x⊤b,x∈Rn

where Q=Q⊤>0\boldsymbol{Q}=\boldsymbol{Q}^{\top}>0Q=Q⊤>0. Our first search direction from an initial point x(0)\boldsymbol{x}^{(0)}x(0) is in the direction of steepest descent; that is,
