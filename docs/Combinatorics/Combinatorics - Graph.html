Graph Theory

Eulerian Trails​

Definition

If the graph GGG has the property that for any two vertices xxx and yyy, one can find a path from xxx and yyy, then we say that GGG is a connected graph.

Definition

If a graph GGG has no loops and has no multiple edges between the same pair of points, then we will say that GGG is a simple graph.

Definition

A sequence of distinct edges e1e2⋯eke_1e_2 \cdots e_ke1​e2​⋯ek​ is called a trail if we can take a continuous walk in our graph. A walk is like a trail, except that all edges do not need to be distinct.

If a trail uses all edges of GGG, then we call it an Eulerian trail. If a trail does not touch any vertex twice, then we call it a path.

Theorem​

A connected graph GGG has a closed Eulerian trail if and only if all vertices of GGG have an even degree.

Proof​

First, we prove the "only if" part, that is, we show that if GGG has a closed Eulerian trail, then all vertices of GGG must have an even degree. Indeed, when we take the closed Eulerian trail WWW, we visit each vertex a certain number of times. Let AAA be a vertex that was not where WWW started, and assume we visited AAA exactly aaa times. This means we entered AAA exactly aaa times, and we left AAA exactly aaa times. As we assumed WWW was a trail, we had to do this using different edges, so we used 2a2 a2a edges. On the other hand, WWW contains all edges of GGG, so AAA cannot have any additional edges, therefore the degree of AAA is 2a2 a2a. This shows that the degree of any vertex other than the starting point SSS of WWW is even. Finally, note that SSS is not only the starting point of WWW, but also the endpoint, so if we visit SSS exactly ttt times between the start and the end of WWW, then we use 1+2t+1=2(t+1)1+2 t+1=2(t+1)1+2t+1=2(t+1) edges. Therefore, the degree of SSS is 2(t+1)2(t+1)2(t+1), and our claim is proved.

Now let us assume that all vertices of GGG have an even degree and prove that GGG has a closed Eulerian trail. Take any vertex SSS, and start walking along an edge e1e_1e1​, to the other endpoint A1A_1A1​ of that edge, then walk along any new edge e2e_2e2​ that starts in A1A_1A1​. Continue this way, using new (previously unused) edges at each step, until a closed trail C1C_1C1​ is formed. As GGG is finite, such a closed trail will always be formed. The first closed trail will be formed when we first revisit a vertex already visited. We cannot get stuck at some vertex before completing a closed trail as each vertex has an even degree, so each time we enter a vertex, we can also leave it, except possibly the initial vertex. If C1=GC_1=GC1​=G, then we are done. If not, then choose a vertex VVV in C1C_1C1​ so that C1C_1C1​ does not contain all edges adjacent to VVV.

The alert reader can ask now how we know that there is such a vertex VVV. Let us assume that there is not. As C1C_1C1​ contains fewer edges than GGG, and supposedly C1C_1C1​ contains all edges adjacent to all vertices it contains, there must be a vertex AAA that is not in C1C_1C1​. However, GGG is a connected graph, so there must be a path connecting AAA to any vertex in C1C_1C1​. Start walking on this path from AAA to any given vertex of C1C_1C1​. When you reach C1C_1C1​ the first time, you will reach it in a vertex VVV that is in C1C_1C1​, but not all the edges adjacent to it are in C1C_1C1​. Indeed, the one that has just ended in VVV is not. This proves by contradiction that such a vertex VVV always exists. Illustrated here:

Let us now remove all edges of C1C_1C1​ from GGG. We get a graph in which again all vertices have an even degree. Starting at VVV, let us take another closed trail C2C_2C2​ in the remaining graph. We can then unite C1C_1C1​ and C2C_2C2​ into one closed trail in GGG. Indeed, if we start walking by C1C_1C1​, we can stop at VVV, walk through C2C_2C2​, then complete our trail by using the remaining part of C1C_1C1​. If the new trail C1∪C2C_1 \cup C_2C1​∪C2​ contains all edges of GGG, we are done. If not, then let us omit C1∪C2C_1 \cup C_2C1​∪C2​ from GGG, and find a new closed trail C3C_3C3​ in the remaining graph.

As GGG has a finite number of edges, this procedure has to stop after a finite number of steps. Therefore, after a finite number of steps, C1∪C2∪⋯∪CkC_1 \cup C_2 \cup \cdots \cup C_kC1​∪C2​∪⋯∪Ck​ will be a closed trail containing all edges of GGG.

Hamiltonian Cycles​

Definition

A cycle in a graph is a closed trail that does not touch any vertex twice, except, the initial vertex, which must also be the ending vertex. A cycle that includes all vertices of a graph is called a Hamiltonian path.

Theorem​

Let n≥3n \geq 3n≥3, let GGG be a simple graph on nnn vertices, and let us assume that all vertices in GGG are of degree at least n/2n / 2n/2. Then GGG has a Hamiltonian cycle.

Proof​

Note that it follows from the conditions that GGG is connected. Indeed, if GGG had more than one component, then it would have a component of less than n/2n / 2n/2 vertices, and vertices in that component would have a degree less than n/2n / 2n/2.

Let us assume that GGG does not have a Hamiltonian cycle. Let us add new edges to GGG as long as we can without creating a Hamiltonian cycle. When we stop, we have a graph G′G^{\prime}G′ in which all vertices have a degree at least n/2n / 2n/2, there is no Hamiltonian cycle, but adding any new edge would create a Hamiltonian cycle.

Let PPP be a path of maximum length in G′G^{\prime}G′. We claim that PPP contains all vertices of G′G^{\prime}G′. Indeed let xxx and yyy be two vertices in G′G^{\prime}G′ that are not connected by an edge. Directed Graphs​

Definition

A graph in which each edge is assigned a direction is called a directed graph or digraph.

Theorem 1​

A directed graph GGG has a closed Eulerian trail if and only if it is balanced and strongly connected.

Proof​

First, we prove that these conditions are necessary. As a closed Eulerian trail WWW leaves each vertex as many times as it enters that vertex, GGG must be balanced. Similarly, WWW provides a trail from any vertex to any vertex, so GGG is strongly connected.

Isomorphisms​

Definition

We say that graphs GGG and HHH are isomorphic if there is a bijection fff from the vertex set of GGG onto that of HHH so that the number of edges between any pair of vertices XXX and YYY of GGG is equal to the number of edges between vertices f(X)f(X)f(X) and f(Y)f(Y)f(Y) of HHH. The bijection fff is called an isomorphism.
