Permutations (Definition)​

The arrangement of different objects into a linear order using each object exactly once is called a permutation of these objects. The number n×(n−1)×(n−2)⋯2×1n \times (n-1) \times (n-2) \cdots 2 \times 1n×(n−1)×(n−2)⋯2×1 of all permutations of nnn objects is called nnn factorial, and is denotes by n!n!n!.

Theorem 1​

The number of all permutations of an n-element set is n!n!n!.

Strings over a Finite Alphabet​

Theorem​

The number of kkk-digit strings over an nnn-element alphabet is nkn^knk.

Proof​

We can choose the first digit in nnn different ways. Then, we can choose the second digit in n different ways as well since we are allowed to use the same digit again (unlike in case of permutations). Similarly, we can choose the third, fourth, etc., kth element in n different ways. We can make all these choices independently from each other, so the total number of choices is nkn^knk.

Definition (Injection and Surjection)

Let f:X→Yf: X \rightarrow Yf:X→Y be a function. If fff satisfies criterion (1) of the bijection definition, then we say that fff is one-to-one or injective, or is an injection. If fff satisfies criterion (2), then we say that fff is onto or surjective, or is a surjection.

Proposition​

Let XXX and YYY be two finite sets. If there exists a bijection fff from XXX onto YYY, then XXX and YYY have the same number of elements.

Proof​

The bijection fff matches elements of XXX to elements of YYY, in other words it creates pairs with one element from XXX and one from YYY in each pair. If fff created mmm pairs, then both XXX and YYY have mmm elements.
