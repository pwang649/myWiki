Halfspaces​

The first hypothesis class we consider is the class of halfspaces, designed for binary classification problems, namely, X=Rd\mathcal{X}=\mathbb{R}^dX=Rd and Y={−1,+1}\mathcal{Y}=\{-1,+1\}Y={−1,+1}. The class of halfspaces is defined as follows:

HSd=sign⁡∘Ld={x↦sign⁡(hw,b(x)):hw,b∈Ld}.H S_d=\operatorname{sign} \circ L_d=\left\{\mathbf{x} \mapsto \operatorname{sign}\left(h_{\mathbf{w}, b}(\mathbf{x})\right): h_{\mathbf{w}, b} \in L_d\right\} .HSd​=sign∘Ld​={x↦sign(hw,b​(x)):hw,b​∈Ld​}.

In other words, each halfspace hypothesis in HSdH S_dHSd​ is parameterized by w∈\mathbf{w} \inw∈ Rd\mathbb{R}^dRd and b∈Rb \in \mathbb{R}b∈R and upon receiving a vector x\mathbf{x}x the hypothesis returns the label sign⁡(⟨w,x⟩+b)\operatorname{sign}(\langle\mathbf{w}, \mathbf{x}\rangle+b)sign(⟨w,x⟩+b)

To illustrate this hypothesis class geometrically, it is instructive to consider the case d=2d=2d=2. Each hypothesis forms a hyperplane that is perpendicular to the vector w\mathbf{w}w and intersects the vertical axis at the point (0,−b/w2)\left(0,-b / w_2\right)(0,−b/w2​). The instances that are "above" the hyperplane, that is, share an acute angle with w\mathbf{w}w, are labeled positively. Instances that are "below" the hyperplane, that is, share an obtuse angle with w\mathbf{w}w, are labeled negatively.

Perceptron for Halfspaces​

A different implementation of the ERM rule is the Perceptron algorithm of Rosenblatt (Rosenblatt 1958). The Perceptron is an iterative algorithm that constructs a sequence of vectors w(1),w(2),…\mathbf{w}^{(1)}, \mathbf{w}^{(2)}, \ldotsw(1),w(2),…. Initially, w(1)\mathbf{w}^{(1)}w(1) is set to be the all-zeros vector. At iteration ttt, the Perceptron finds an example iii that is mislabeled by w(t)\mathbf{w}^{(t)}w(t), namely, an example for which sign⁡(⟨w(t),xi⟩)≠yi\operatorname{sign}\left(\left\langle\mathbf{w}^{(t)}, \mathbf{x}_i\right\rangle\right) \neq y_isign(⟨w(t),xi​⟩)=yi​. Then, the Perceptron updates w(t)\mathbf{w}^{(t)}w(t) by adding to it the instance xi\mathbf{x}_ixi​ scaled by the label yiy_iyi​. That is, w(t+1)=w(t)+yixi\mathbf{w}^{(t+1)}=\mathbf{w}^{(t)}+y_i \mathbf{x}_iw(t+1)=w(t)+yi​xi​. Recall that our goal is to have yi⟨w,xi⟩>0y_i\left\langle\mathbf{w}, \mathbf{x}_i\right\rangle>0yi​⟨w,xi​⟩>0 for all iii and note that

Hence, the update of the Perceptron guides the solution to be "more correct" on the iii 'th example.
