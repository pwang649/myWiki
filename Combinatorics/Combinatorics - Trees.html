Minimally Connected Graphs​

Theorem 1​

Let GGG be a connected simple graph on nnn vertices. Then the following are equivalent.

(1) GGG is minimally connected, that is, if we remove any edge of GGG, then the obtained graph G′G^{\prime}G′ will not be connected.

(2) GGG does not contain a cycle.

Definition

A connected simple graph GGG satisfying either, and therefore, both, criteria of Theorem 1 is called a tree.

Proof​

(1) ⇒\Rightarrow⇒ (2) Let us assume that GGG is minimally connected, but it contains a cycle CCC. Remove the edge aba bab of CCC. We claim that GGG is still connected. Indeed, let xxx and yyy be two vertices in GGG. As GGG is connected, GGG contains a path ppp from xxx to yyy. If ppp does not contain the edge aba bab, then it still connects xxx and yyy. If ppp contains aba bab, then let us replace aba bab by the other (longer) arc aba bab, to get a new walk from xxx to yyy. As there is a walk from xxx to yyy in G′G^{\prime}G′, there must also be a path. Therefore, G′G^{\prime}G′ is connected, which is a contradiction.

(2) ⇒\Rightarrow⇒ (1) We prove that the opposite of (1) implies the opposite of (2). That will suffice, because it will imply that if (2) holds, the opposite of (1) cannot hold as that would imply the opposite of (2), therefore (1) has to hold. So "(2) implies (1)" will follow. Let us assume that GGG is not minimally connected. That means that there is an edge in GGG, say ABA BAB, so that G′=G−{AB}G^{\prime}=G-\{A B\}G′=G−{AB} is still connected. Then there is a path PPP from BBB to AAA in G′G^{\prime}G′. However, AB∪PA B \cup PAB∪P must then be a cycle in GGG as it defines a path that starts in AAA and ends in A. So GGG contains a cycle.

Corollary​

A connected graph HHH is a tree if and only if for each pair of vertices (x,y)(x, y)(x,y), there is exactly one path joining xxx and yyy.

Theorem 2​

All trees on nnn vertices have n−1n-1n−1 edges. Conversely, all connected graphs on nnn vertices with exactly n−1n-1n−1 edges are trees.

Proof​

We use induction on nnn. If n=1n=1n=1, the statement is trivially true as a 1-vertex cycle-free graph has no edges. Let us assume that the statement is true for trees on nnn vertices. Let TTT be a tree on n+1n+1n+1 vertices. Find a leaf lll in TTT (the previous lemma ensures the existence of two leaves), then delete lll and the only edge eee adjacent to it from TTT, to get a new tree T′T^{\prime}T′. (Note that T′T^{\prime}T′ is always a tree as it is connected and cycle-free.) This new tree T′T^{\prime}T′ has nnn vertices, so by the induction hypothesis, it has n−1n-1n−1 edges. But then T=T′∪eT=T^{\prime} \cup eT=T′∪e has nnn edges, and the Theorem is proved.

Proposition​

Let FFF be a forest on nnn vertices with kkk connected components. Then FFF has n−kn-kn−k edges.

Proof​

By Theorem 2, the number of vertices exceeds that of edges by one in each connected component, and the proof follows.

Minimum-weight Spanning Trees​

Kruskal's Greedy Algorithm
  1. Create a forest FFF (a set of trees), where each vertex in the graph is a separate tree
  2. Create a sorted set SSS containing all the edges in the graph
  3. While SSS is nonempty and FFF is not yet spanning
    • Remove an edge with minimum weight from SSS
    • If the removed edge connects two different trees then add it to the forest FFF, combining two trees into a single tree

Lemma​

Let FFF and F′F^{\prime}F′ be two forests on the same vertex set VVV, and let FFF have less edges than F′F^{\prime}F′. Then F′F^{\prime}F′ has an edge e that can be added to FFF so that the obtained graph F∪eF \cup eF∪e is still a forest.

Proof​

Let us assume that there is no such edge e∈F′e \in F^{\prime}e∈F′. Then adding any edge of F′F^{\prime}F′ to FFF would create a cycle in FFF. So all edges of F′F^{\prime}F′ are between two vertices of the same component of FFF. Therefore, F′F^{\prime}F′ has at least as many components as FFF. This is a contradiction, however, as we know that a forest on nnn vertices and with kkk components has n−kn-kn−k edges, so if F′F^{\prime}F′ has more edges than FFF, it must have less components. Now we are in a position to prove the main result of this section.
