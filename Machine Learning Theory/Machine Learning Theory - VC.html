Definition (VC Dimension)​

The VC-dimension of a hypothesis class HHH, denoted VCdim(H)VCdim(H)VCdim(H), is the maximal size of a set C⊂XC \subset XC⊂X that can be shattered by HHH. If HHH can shatter sets of arbitrarily large size we say that HHH has infinite VC-dimension.

Theorem​

Let HHH be a class of infinite VC-dimension. Then, H is not PAC learnable.

Proof​

Since HHH has in infinite VC-dimension, for any training set size m, there exists a shattered set of size 2m, and the claim follows by the previous corollary.
